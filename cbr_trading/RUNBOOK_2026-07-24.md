# CBR live runbook — 24 July 2026

The Bank of Russia press release is scheduled for 13:30 Moscow time
(12:30 Budapest time).

## Confirmed trading profile

- Previous key rate: 14.25%.
- Active rule: `98 / cbr_decrease_fast`.
- Market: “Will the Bank of Russia decrease the key rate after the July
  Meeting?”
- If the new rate is below 14.25%: BUY YES, quantity 100, limit 0.20.
- If the new rate is 14.25% or above: BUY NO, quantity 100, limit 0.20.
- Every order is post-only GTC with maximum notional 20.
- A fresh order book is checked immediately before submission. The order is
  skipped if 0.20 would cross the current ask.
- A persistent database claim prevents a second submission for the same
  release, rule, action, account, condition, quantity, and price.
- Telegram is called only after order processing finishes.

- maximum notional 20;
- post-only enabled and the master key present.

## Start

Start around 12:20 Budapest time so the account, both outcomes, and database
ledger are warmed before publication:

```powershell
.\.venv\Scripts\python.exe -m cbr_trading
```

Do not proceed unless the log contains:

```text
CBR live executor warmed before polling rules=1 accounts=1 outcomes=2
```

Leave the terminal and network connection running. Use `Ctrl+C` only to stop
before the release. After detection, inspect the final JSON, Telegram message,
Polymarket orders, and the execution-ledger row before considering a restart.
